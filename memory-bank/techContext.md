# Tech Context

## Technologies Used
- Flutter SDK >=3.0.0 (web target).
- Dependencies: provider, firebase_core/firestore, intl, uuid, table_calendar, fl_chart.

## Development Setup
- `flutter pub get && flutter run -d chrome`.

## Technical Constraints
- Web-first; Firebase rules for security.
- Auth system implemented; no seeded data.

## Dependencies
- provider: State management.
- firebase_core/firestore: Backend.
- intl/uuid: Data handling.
- table_calendar/fl_chart: UI components.

## Tool Usage Patterns
- Flutter CLI for build/run; Firebase console for data.
