# Project Brief: CTP Gravure Overtime Tracker

## Core Requirements & Goals
- Browser-first Flutter Web app for overtime management.
- Key features: Split-screen entry/list for overtime/jobs, job overlap analysis, custom times, CTP color themes (Badenia green, Wifag orange, Aurora blue).
- Firebase Firestore for real-time data (jobs, overtime entries).
- Bottom nav: Dashboard, Overtime, Jobs, Analysis, Calendar.
