# Product Context

## Why This Project Exists
- Solve manual overtime tracking inefficiencies at CTP Gravure.
- Automatic job overlap calculation for accurate approvals.

## How It Works
- Add/edit overtime/jobs via intuitive split-screen.
- Analysis screen: Match jobs then compute overlaps.
- Real-time sync via Firebase.

## UX Goals
- Fast entry (duplicate/edit), dark/light toggle, mobile-responsive web.
